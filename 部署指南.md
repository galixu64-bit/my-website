# 🚀 网站部署指南 - 让全世界找到你的网站

## 📋 方案一：GitHub Pages（推荐，完全免费）

### 步骤 1：创建 GitHub 仓库
1. 访问 https://github.com
2. 注册/登录账号
3. 点击右上角 "+" → "New repository"
4. 仓库名称：`my-website` 或 `dragbit`
5. 选择 **Public**（公开）
6. 不要勾选 "Initialize with README"
7. 点击 "Create repository"

### 步骤 2：上传代码到 GitHub
1. 下载并安装 [Git](https://git-scm.com/downloads)
2. 打开命令提示符（CMD）或 PowerShell
3. 进入你的网站目录：
   ```bash
   cd C:\Users\shiji\my-website
   ```
4. 执行以下命令：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/my-website.git
   git push -u origin main
   ```
   （将 `你的用户名` 替换为你的 GitHub 用户名）

### 步骤 3：启用 GitHub Pages
1. 在 GitHub 仓库页面，点击 **Settings**
2. 左侧菜单找到 **Pages**
3. **Source** 选择：`main` 分支，`/` 根目录
4. 点击 **Save**
5. 等待 1-2 分钟，访问：`https://你的用户名.github.io/my-website`

### 步骤 4：自定义域名（可选）
1. 购买域名（推荐：Namecheap, GoDaddy, 或国内域名商）
2. 在 GitHub Pages 设置中添加自定义域名
3. 在域名 DNS 设置中添加 CNAME 记录

## 📋 方案二：Netlify（最简单，免费且功能强大）

### 步骤 1：注册 Netlify
1. 访问 https://www.netlify.com
2. 点击 "Sign up" 注册（可用 GitHub 账号登录）

### 步骤 2：部署网站
#### 方法 A：直接拖拽部署（最简单）
1. 登录 Netlify
2. 进入 Dashboard
3. 将整个 `my-website` 文件夹拖到页面上
4. 等待部署完成（约 30 秒）
5. 你会得到一个免费的网址：`https://随机名称.netlify.app`

#### 方法 B：连接 GitHub（推荐，自动更新）
1. 在 Netlify Dashboard 点击 "Add new site" → "Import an existing project"
2. 选择 GitHub，授权并选择你的仓库
3. 点击 "Deploy site"
4. 每次你推送代码到 GitHub，网站会自动更新！

### 步骤 3：自定义域名
1. 在 Netlify 网站设置中点击 "Domain settings"
2. 点击 "Add custom domain"
3. 输入你的域名并按照提示配置 DNS

1. 访问 https://www.netlify.com 注册

## ⚠️ 注意事项

1. **隐私保护**：不要在代码中硬编码敏感信息（API密钥等）
2. **备份**：定期备份你的代码和数据库
3. **性能**：使用 CDN 加速，优化图片大小
4. **安全**：定期更新依赖，使用 HTTPS
5. **监控**：使用 Google Analytics 等工具监控访问量
