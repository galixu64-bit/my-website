# 🔍 网站显示空白问题诊断指南

## 可能的原因

### 1. **需要登录才能查看内容** ⚠️
网站设置了"必须登录才能查看"的限制。如果你没有登录，页面只会显示登录提示。

**解决方法：**
- 访问 `http://你的IP地址/login.html` 登录
- 或者使用演示账号登录（查看登录页面的提示信息）

### 2. **JavaScript 文件加载失败**
打开浏览器开发者工具（按 `F12`），查看 Console（控制台）是否有红色错误信息。

**常见错误：**
- `Failed to load resource: resources.json` - 资源文件加载失败
- `i18n is not defined` - 翻译文件加载失败
- `CORS error` - 跨域问题（本地服务器配置问题）

### 3. **服务器路径配置问题**
如果通过 IP 访问，确保：
- 服务器指向正确的文件夹（`my-website`）
- 访问的是 `http://IP地址/index.html` 或 `http://IP地址/`

## 🔧 快速诊断步骤

### 步骤 1：打开浏览器开发者工具
1. 按 `F12` 键
2. 点击 **Console（控制台）** 标签
3. 查看是否有红色错误信息

### 步骤 2：检查网络请求
1. 在开发者工具中点击 **Network（网络）** 标签
2. 刷新页面
3. 查看以下文件是否加载成功（状态码应该是 200）：
   - `index.html`
   - `script.js`
   - `auth.js`
   - `translations.js`
   - `resources.json`
   - `style.css`

### 步骤 3：检查登录状态
在 Console 中输入以下命令：

```javascript
localStorage.getItem('currentUser')
```

- 如果返回 `null` 或空，说明**没有登录**
- 如果有内容，说明已登录

### 步骤 4：尝试强制登录
如果显示未登录，访问登录页面：
```
http://你的IP地址/login.html
```

登录后，再返回主页查看。
